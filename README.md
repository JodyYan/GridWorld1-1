# Grid Map Generator (網格地圖產生器)

這是一個基於 HTML, CSS (CSS Grid) 和 JavaScript (Vanilla JS) 開發的互動式網格地圖產生器。
主要用於動態生成 $n \times n$ 的網格，並提供起點、終點與障礙物的嚴格狀態設定指令與防呆功能。

## 🌟 專案 Demo

👉 **[點此前往 Demo 網頁](https://jodyyan.github.io/GridWorld1-1/)**

## ✨ 功能特色

- **動態網格生成**：使用者可輸入介於 5 到 9 之間的數字 $n$，點擊按鈕後動態產生 $n \times n$ 的網格。
- **起點與終點設定**：
  - 第 1 次點擊網格：自動設定為「起點」（綠色狀態）。
  - 第 2 次點擊網格：自動設定為「終點」（紅色狀態）。
- **障礙物設定與防呆機制**：
  - 第 3 次起的點擊：會將網格設定為「障礙物」（灰色狀態）。
  - 精確限制：一盤遊戲中**只能設定剛好 $n-2$ 個障礙物**。
  - 防呆保護：若嘗試設定超過限制的障礙物，系統將跳出 Alert 警告視窗，且不會覆蓋已有狀態的網格。

## 🚀 技術堆疊

- **前端**：HTML5, CSS3, Vanilla JavaScript (DOM 操作與狀態機管理)
- **排版系統**：CSS Grid (確保 $n \times n$ 網格完美對齊與正方形渲染)
- **部署**：GitHub Pages (純靜態網頁託管)
- *(附註：本專案在最初開發時，曾使用 Python Flask 架設本機開發伺服器，後續因應 GitHub Pages 需求已抽出成為純靜態架構)*

## 📂 專案重點檔案

- `index.html`：網頁主結構與 UI 控制介面。
- `static/style.css`：網格排版 (CSS Grid) 與狀態的視覺樣式處理。
- `static/script.js`：網格產生邏輯、點擊事件監聽、狀態計數防呆處理。
- `log.md`：開發與功能驗證的對話過程紀錄。
